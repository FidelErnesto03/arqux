# BLP-001: Restaurar pulse.compact — regresión BLP-016

---

## §1: Planteamiento del Problema

BLP-016 eliminó accidentalmente `compact_session_pulse` de `src/arqux/pulse.py` durante el refactor `write_text → cortex_write`. La función existía y 18 tests pasaban (auditoría Heimdall 2026-07-13: "sólido, 18/18 tests verdes"). Hoy los 18 tests fallan con:

```
ImportError: cannot import name 'compact_session_pulse' from 'arqux.pulse'
```

**Evidencia:**
- `from arqux.pulse import compact_session_pulse` → ImportError
- `pytest tests/test_pulse_compact.py -v` → 18 failed
- `git log --all --oneline -- src/arqux/pulse.py` → commit bc675cf la contiene
- BLP-016.md no menciona pulse.compact en su alcance (§2, §6)

---

## §2: Objetivo

Restaurar la función `compact_session_pulse` en `src/arqux/pulse.py` desde el commit `bc675cf` (CYCLE-02 cierre formal) donde existía y funcionaba correctamente con 18 tests verdes. No es implementar desde cero — es recuperar código perdido por un refactor no intencional.

---

## §3: Precondiciones

- Acceso a `src/arqux/pulse.py` (estado actual, post-BLP-016)
- Acceso a `tests/test_pulse_compact.py` (18 tests, actualmente rotos)
- Acceso a git history: commit `bc675cf` contiene la función original
- Bloqueante: si `bc675cf` no es accesible, usar `git log --all` para encontrar la función

---

## §4: Principio Rector

**Restaurar, no reimplementar.** La funcionalidad ya fue diseñada, implementada, testeada y aprobada. El objetivo es recuperar el código exacto, no rediseñarlo. Si hay conflictos con imports post-BLP-016, adaptar solo los imports — nunca la lógica.
