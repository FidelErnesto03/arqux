# BLP-003: Migrar skill.py:210 — write_text → cortex_write

---

## §1: Planteamiento del Problema

**skill.py:210** usa `write_text()` para escribir archivos .cortex, violando directamente **AXM:codec_cortex_writer**: _"All .cortex file writes MUST go through CODEC-CORTEX. Direct write_text() is FORBIDDEN."_

BLP-016 migró 4 de 5 archivos objetivo (project.py, session.py, workspace.py, _helpers.py) a `cortex_write`/`crud_add` exitosamente. **skill.py:210 quedó sin migrar.**

```python
# skill.py:210 — código actual (violación)
dst.write_text(cortex_content, encoding="utf-8")
```

---

## §2: Objetivo

Reemplazar `dst.write_text(cortex_content, encoding="utf-8")` en `skill.py:210` por `cortex_write(path=str(dst), content=cortex_content)` usando el writer CODEC-CORTEX ya existente en `src/arqux/state.py`.

---

## §3: Precondiciones

- `cortex_write` existe en `src/arqux/state.py` (verificado en BLP-016)
- `skill.py` importa desde `..state` (verificar que cortex_write esté disponible)
- Tests existentes para skill.py

---

## §4: Principio Rector

**CODEC-CORTEX como writer exclusivo.** Ningún handler debe escribir archivos .cortex fuera del pipeline CODEC-CORTEX. La migración de BLP-016 está incompleta hasta que skill.py:210 se migre.

---

## §5: Contexto

```
skill.py:210 (VIOLACIÓN)          skill.py:210 (CORRECTO)
─────────────────────────         ────────────────────────
dst.write_text(                   cortex_write(
  cortex_content,                   path=str(dst),
  encoding="utf-8"                  content=cortex_content
)                                 )
```

---

## §6: Alcance y Exclusiones

**Dentro del alcance:**
- Modificar `skill.py:210` para usar `cortex_write`
- Verificar que el comportamiento es idéntico

**Fuera del alcance:**
- Otros usos de write_text en skill.py (si existen)
- Modificar cortex_write
- Nuevos tests (los existentes deben pasar)

---

## §7: Reglas Obligatorias

1. Solo modificar la línea de escritura — no cambiar lógica
2. Verificar que cortex_write está importado
3. Si no está importado, agregar import

---

## §8: Diseño Técnico

**Cambio:**
```python
# Antes (skill.py:210)

# Después
from ..state import cortex_write  # verificar si ya está importado

```
