## §8: Diseño Técnico

El parche introduce 3 capas nuevas:
- **security.py:** HMAC-SHA256 para identidad, SHA-256 + Ed25519 para integridad de .cortex
- **concurrency.py:** file locking con fcntl + placeholders para IDs atomicos
- **permissions.py:** enum Role con enforcement real, HMAC required handlers

Modifica handlers existentes:
- cycle.py: agrega mature_cycle (draft->ready) y constants CYCLE_DRAFT/READY
- sync.py: fix GAP-001 usando _resolve_brain_path
- cortex.py: HMAC verification en record_lesson_handler
- blueprint.py, state.py: delegan a concurrency para IDs seguros
